ClassSync

Extension Instructions

  • Step 1: Clone extension repository from ClassSync GitHub
  • Step 2: Navigate to Chrome Extensions
  • Step 3: Select 'Load Unpacked', then select the 'extension' folder from the cloned repository
  • Step 4: Navigate to Canvas Settings and select '+ New Access Token'
  • Step 5: Enter your new Canvas API Token into the ClassSync extension and select 'Save'
